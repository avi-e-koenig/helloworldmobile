# 🤖 AI CONTEXT MAINTENANCE RULES

## **Primary Rules for AI Context Upkeep**

### **RULE 1: Always Update Core Index Files**
**When making ANY changes to the project:**
- ✅ Update `AI-CONTEXT-INDEX.md` with current status
- ✅ Update `TECHNICAL-REFERENCE.md` with new configurations
- ✅ Update `STARTUP-GUIDE.md` if commands change
- ✅ Update `SEARCH-INDEX.md` with new keywords

### **RULE 2: Maintain File Priority Hierarchy**
**Always keep files in this priority order:**
1. **HIGH PRIORITY** (Essential for AI tools):
   - `AI-CONTEXT-INDEX.md`
   - `STARTUP-GUIDE.md`
   - `TECHNICAL-REFERENCE.md`
   - `TROUBLESHOOTING-GUIDE.md`
   - `SEARCH-INDEX.md`

2. **MEDIUM PRIORITY** (Important reference):
   - `package.json`
   - `App.tsx`
   - `.vscode/launch.json`
   - `app.json`
   - `tsconfig.json`

3. **LOW PRIORITY** (Detailed documentation):
   - Directory-based documentation files
   - Code examples
   - Enhancement guides

### **RULE 3: Synchronize Information Across Files**
**When updating any file:**
- ✅ Ensure consistency across all related files
- ✅ Update cross-references and links
- ✅ Verify command accuracy in all guides
- ✅ Check that technical details match implementation

### **RULE 4: Preserve AI-Optimized Structure**
**Never break the flat file structure:**
- ✅ Keep essential files in root `ai-context/` directory
- ✅ Maintain quick access to core information
- ✅ Avoid deep nesting for critical files
- ✅ Ensure AI tools can find information quickly

## **Specific Maintenance Tasks**

### **When Adding Dependencies**
1. Update `package.json` (automatic)
2. Update `TECHNICAL-REFERENCE.md` dependencies section
3. Update `AI-CONTEXT-INDEX.md` tech stack
4. Update `SEARCH-INDEX.md` keywords if new technology
5. Update `TROUBLESHOOTING-GUIDE.md` if new potential issues

### **When Changing Scripts**
1. Update `package.json` scripts (automatic)
2. Update `STARTUP-GUIDE.md` commands
3. Update `TECHNICAL-REFERENCE.md` scripts section
4. Update `AI-CONTEXT-INDEX.md` commands reference
5. Update VS Code launch configurations if needed

### **When Modifying App.tsx**
1. Update `App.tsx` (automatic)
2. Update `TECHNICAL-REFERENCE.md` implementation section
3. Update `AI-CONTEXT-INDEX.md` current implementation
4. Update `SEARCH-INDEX.md` components section
5. Update code examples if applicable

### **When Fixing Issues**
1. Document the fix in `TROUBLESHOOTING-GUIDE.md`
2. Update `AI-CONTEXT-INDEX.md` fixes applied section
3. Update `TECHNICAL-REFERENCE.md` key fixes section
4. Update `SEARCH-INDEX.md` issues fixed section

### **When Adding Features**
1. Update `AI-CONTEXT-INDEX.md` key features
2. Update `TECHNICAL-REFERENCE.md` implementation
3. Update `SEARCH-INDEX.md` keywords
4. Update relevant directory documentation
5. Update `next-steps/enhancements.md` if applicable
