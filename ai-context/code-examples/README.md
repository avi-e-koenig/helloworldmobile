# Code Examples

This folder contains various code examples for implementing React Native Hello World with your existing Expo project and React Native Paper.

## Files

### `App.tsx`
Basic Hello World implementation to replace your current `App.tsx`:
- Material Design 3 components
- Centered layout
- Simple button interaction
- Transforms your basic Expo template into Material Design

### `advanced-example.tsx`
Extended example featuring:
- Theme toggle (light/dark mode)
- Text input component
- Multiple buttons with different modes
- Switch component
- Card-based layout
- ScrollView for better content organization

## Usage

1. Copy the desired example code
2. Replace your current `App.tsx` file with the example
3. Install React Native Paper: `npm i react-native-paper`
4. Run your Expo development server: `npm run start`

## Component Overview

### Basic Components Used
- `PaperProvider` - Theme context provider
- `SafeAreaView` - Safe area wrapper
- `Text` - Material Design typography
- `Button` - Material Design buttons

### Advanced Components Used
- `Card` - Material Design cards
- `TextInput` - Text input fields
- `Switch` - Toggle switches
- `ScrollView` - Scrollable content
- `MD3LightTheme` / `MD3DarkTheme` - Theme variants
