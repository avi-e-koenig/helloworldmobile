## Quick Commands

### **🚀 Start Everything (Recommended):**
```bash
npm run web
```
Starts both web and mobile platforms simultaneously.

### **📱 Mobile Only:**
```bash
npm run start
```

### **🧹 Cleanup:**
```bash
npm run cleanup
```

### **🔧 VS Code:**
Press **F5** and select your desired launch configuration.

## Access URLs

- **Web**: `http://localhost:3002`
- **Mobile**: Scan QR code with Expo Go
- **Metro Bundler**: Port 3001
